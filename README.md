# Wecomic public website

Static public website for Wecomic apps.

- `index.html`: developer app directory for WeYoung and Rage Daily
- `weyoung.html`: WeYoung product landing page
- `privacy.html`: Rage Daily bilingual privacy policy
- `support.html`: Rage Daily bilingual support page
- `app-ads.txt`: AdMob authorized seller record

The site intentionally contains no application source code or private account data.
